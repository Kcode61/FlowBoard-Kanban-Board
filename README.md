# FlowBoard

FlowBoard é um Kanban board simples para organização de tarefas, criado como um projeto de estudo e prática.  
Esse foi meu **primeiro projeto usando Next.js**, então o foco principal foi aprender a ferramenta e construir algo funcional ao mesmo tempo.

A ideia do projeto é manter tudo o mais simples possível, sem overengineering, apenas o essencial para funcionar bem.

---

## Acesso ao projeto

Você pode acessar o projeto online aqui: https://flowboard-kanban-board.pages.dev


---

## Sobre o projeto

O FlowBoard segue o conceito básico de um quadro Kanban, permitindo visualizar tarefas em diferentes etapas do fluxo de trabalho.

Não é um projeto comercial nem algo super complexo. Ele foi criado com o objetivo de aprender, errar, refatorar e evoluir com o tempo.

Muita coisa aqui pode (e provavelmente vai) mudar.

---

## Tecnologias utilizadas

- Next.js
